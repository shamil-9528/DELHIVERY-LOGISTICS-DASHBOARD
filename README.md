## 📌 Project Description

This project focuses on analyzing **logistics delivery performance** using the Delhivery logistics dataset and building an **interactive Power BI dashboard** to monitor operational efficiency.

The dashboard provides insights into **delivery delays, route efficiency, trip performance, and logistics trends** across different routes and cities. It helps identify inefficiencies and supports data-driven decision making for improving delivery operations.

---

## 📈 Key Metrics

- **Total Trips:** 145K  
- **Average Delay:** 203.06  
- **Average Actual Time:** 417  
- **On-Time Delivery Rate:** 18.03%  
- **Late Deliveries:** 119K  

---

## 📊 Dashboard Features

- Delivery status analysis (**Late vs On-Time deliveries**)
- **Route efficiency classification** (Efficient, Average, Inefficient)
- **FTL vs Carting delivery time comparison**
- **Average delay by source city**
- **Average delay by destination city**
- **Trip distribution by route type**
- Interactive visual insights for logistics operations

## ⚙️ Methodology

### Data Preparation
- Imported logistics dataset into **Power BI**

- Handled missing values and inconsistencies

### Data Modeling

- Total Trips
- Average Delay
- Average Delivery Time
- On-Time Delivery Percentage
- Late Delivery Count

### Data Visualization
Designed interactive dashboard using:
- KPI Cards
- Bar Charts

- Pie Charts
